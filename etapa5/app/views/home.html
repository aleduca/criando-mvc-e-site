Bem Vindo

Muito bem vindo ao meu MVC, esse projeto foi criado somente para o propósito de estudo pessoal, nesse projeto irei usar algumas bibliotecas de terceiros para deixar o sistema muito melhor.
Se precisar de um framework, use um dos vários que existem por ai, por ter mais recursos e ser um framework completo.

Nesse projeto eu usei o twig para os templates e o phpactiverecords para o banco de dados, então se você não tiver conhecimento nessas duas ferramentas, é muito importante começar a estuda-las.
